## 📝 背景说明

用户需要通过登录才能访问系统的核心功能。支持邮箱和手机号两种登录方式，后续可扩展第三方登录。

---

## 🔧 实现方案

### 技术选型

| 模块 | 技术 |
|------|------|
| UI 框架 | React + TailwindCSS |
| 表单验证 | React Hook Form + Zod |
| 状态管理 | Zustand |
| API 请求 | Axios |

### 关键步骤

1. 创建 `LoginForm` 组件
2. 实现表单验证 Schema
3. 对接 `/api/auth/login` 接口
4. 处理登录成功/失败状态
