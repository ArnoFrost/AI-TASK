## 📜 AI 协作规范

### 1. 自动归档规则

**触发条件**：每次对话开始时，AI 应检查并执行归档

**归档策略**：
- `tasks/` 仅保留**近3天**的任务文件
- 超过3天的任务自动移至 `archive/YYYY-MM/`
- 进行中任务始终保留（无论时间）

### 2. index.md 精简规则

**展示原则**：
- 仅展示**近3天**的任务列表
- 进行中任务始终展示
- 历史任务通过归档目录查看

### 3. 规范自约束

| 约束项 | 规则 |
|--------|------|
| index.md 行数 | ≤ 150 行 |
| tasks/ 文件数 | ≤ 15 个 |
| 单任务文件大小 | ≤ 30KB |
| 归档检查频率 | 每次对话开始 |

## 📐 任务命名格式

```
{日期}-{序号}_[标签]任务名称.md
```

**示例**：
```
20260203-003_[架构]Skill统一与多IDE适配层演进.md
```

**编号规则**：**全局递增**（跨日期），查看 tasks/ 目录最大编号 +1

## 🏗️ 三层架构

```
┌─────────────────────────────────────────┐
│  IDE 适配层 (templates/{ide}/)           │  ← 薄层：入口文件
├─────────────────────────────────────────┤
│  共享命令层 (templates/shared-commands/) │  ← 中层：命令定义
├─────────────────────────────────────────┤
│  核心能力层 (skills/)                    │  ← 厚层：实际逻辑
└─────────────────────────────────────────┘
```

- **skills/** - 核心能力（agentskills.io 格式）
- **shared-commands/** - 公共命令（IDE 无关）
- **claude/** / **codebuddy/** - IDE 入口适配
