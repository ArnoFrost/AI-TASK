## 📜 AI 协作规范

### 1. 归档规则

- `tasks/` 仅保留**近3天**的任务文件
- 超过3天的任务自动移至 `archive/YYYY-MM/`
- 进行中任务始终保留

### 2. 命名格式

```
{日期}-{序号}_[标签]任务名称.md
```

**编号规则**：**日内递增**（每天从 001 开始），查看 tasks/ 目录当天最大编号 +1

### 3. 隐私说明

本项目的演进任务（`tasks/`、`docs/`）在本地管理，不公开到开源仓库。

仅公开：
- `index.md` - 项目入口模板
- `project.yaml` - 元数据模板
- `README.md` - 协作规范说明
