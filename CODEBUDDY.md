# AI-TASK

> AI 协作任务管理系统 · v1.3.1

## 🔧 快捷命令

| 命令 | 说明 |
|------|------|
| `/task create [标签] 名称` | 创建任务 |
| `/task list` | 列出任务 |
| `/task done 编号` | 标记完成 |
| `/archive [编号]` | 归档任务 |
| `./init-project.sh` | 初始化项目 |
