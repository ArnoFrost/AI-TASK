查看当前项目的 AI-TASK 状态概览。

## 用户输入

$ARGUMENTS

## 参数解析

```
/status [--verbose] [--json]
```

| 参数 | 必需 | 说明 |
|------|------|------|
| `--verbose` | ⚪ | 显示详细信息 |
| `--json` | ⚪ | 以 JSON 格式输出 |
| `--help` | ⚪ | 显示帮助 |

## 执行流程

### 1. 读取项目信息

从 `ai-task/project.yaml` 读取：
- 项目代号 (code)
- 项目名称 (name)
- 技术栈 (tech_stack)
- 状态 (status)
- 关联项目 (related)

### 2. 统计任务

扫描 `ai-task/tasks/` 目录：
- 进行中任务数量
- 已完成任务数量
- 最近更新的任务

扫描 `ai-task/archive/` 目录：
- 归档任务总数
- 按月份分布

### 3. 输出格式

#### 标准输出

```
📊 AI-TASK 状态概览
═══════════════════════════════════════

📁 项目: {NAME} ({CODE})
📍 路径: {PROJECT_PATH}
🔧 技术栈: {TECH_STACK}
📅 创建时间: {CREATED}

───────────────────────────────────────
📋 任务统计
───────────────────────────────────────
  🔄 进行中: {N} 个
  ✅ 已完成: {N} 个
  📦 已归档: {N} 个

───────────────────────────────────────
🕐 最近活动
───────────────────────────────────────
  • {任务名称} - {状态} ({时间})
  • {任务名称} - {状态} ({时间})

───────────────────────────────────────
🔗 关联项目
───────────────────────────────────────
  • {关联项目1}
  • {关联项目2}
```
