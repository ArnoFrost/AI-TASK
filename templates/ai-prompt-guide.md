# AI 协作提示词指南

> 本文档用于告知 AI 如何与你进行任务协作

## 🎯 核心规则

将以下内容在新对话开始时告知 AI：

```
请记住以下 AI 任务管理规则：

1. **任务存储位置**：所有任务存放在 AI-TASK/projects/{PROJECT_CODE}/tasks/

2. **命名格式**：YYYYMMDD-NNN_[标签]任务名称.md
   - 标签可选：[环境][功能][优化][修复][排查][文档][测试][规范][依赖][下线]
   - 编号从 001 开始，每日递增

3. **模板位置**：AI-TASK/templates/task-template.md

4. **项目索引**：每个项目在 projects/ 目录下有独立索引文件

5. **新建任务时**：
   - 使用任务模板
   - 更新对应项目的索引文件
   - 在任务文档中记录项目代号

6. **更新任务时**：
   - 更新进度记录
   - 更新验证结果
   - 完成后更新状态
```

## 📝 常用指令模板

### 创建新任务
```
请创建任务文档：20251219-001_[功能]xxx功能开发.md
项目：{PROJECT_NAME} ({PROJECT_CODE})
任务目标：
1. 目标 1
2. 目标 2
```

### 更新任务进度
```
请更新任务 20251219-001_[功能]xxx功能开发.md
- 当前进度：50%
- 完成内容：xxx
- 下一步：xxx
```

### 标记任务完成
```
请将任务 20251219-001_[功能]xxx功能开发.md 标记为完成
验证结果：
- 验证项 1: ✅ 通过
- 验证项 2: ✅ 通过
```

### 查看任务列表
```
请列出 ai-task/tasks 目录下今日的所有任务
```

### 生成日报
```
请根据今日的任务进展，生成日报
```

### 查看项目任务汇总
```
请查看 {PROJECT_CODE} 项目的任务汇总
```

## 🔗 快捷路径

| 用途 | 路径 |
|------|------|
| 任务目录 | `ai-task/tasks/` |
| 模板目录 | `AI-TASK/templates/` |
| 项目索引 | `AI-TASK/projects/` |
| 总体说明 | `AI-TASK/README.md` |

---

## 💡 最佳实践

1. **每次新对话开始时**：告知 AI 任务管理规则
2. **开始新任务前**：让 AI 创建任务文档
3. **任务进行中**：定期让 AI 更新任务进度
4. **任务完成后**：让 AI 更新验证结果和状态
5. **每日结束时**：让 AI 生成日报汇总
